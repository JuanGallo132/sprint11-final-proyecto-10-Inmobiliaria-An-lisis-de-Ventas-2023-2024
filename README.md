# sprint11-final-proyecto-10-Inmobiliaria-An-lisis-de-Ventas-2023-2024
Dashboard de Business Intelligence para el sector inmobiliario que transforma datos transaccionales en insights estratégicos. Analiza el crecimiento interanual (11,14% en ingresos), eficiencia de canales y comportamiento de clientes entre 2023 y 2024 para optimizar la toma de decisiones ejecutiva.
